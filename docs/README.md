# Portfólio Leonardo Silva

Site estático de portfólio para publicação no GitHub Pages.

## Estrutura

- `index.html`: página principal com dados pessoais e projetos.
- `styles.css`: identidade visual e responsividade.
- `assets/`: imagens do aluno e screenshots dos projetos.

## Como publicar

1. Envie os arquivos para o repositório `leoesilva.github.io`.
2. No GitHub, abra `Settings > Pages`.
3. Selecione a branch principal e a pasta raiz do projeto.
4. Salve e aguarde a publicação.

## O que ainda precisa ser preenchido

- Foto do aluno.
- Cursos de extensão.
- Tecnologias, participação e screenshots de cada projeto.
- Níveis de inglês e espanhol, se quiser detalhar além de "a informar".
